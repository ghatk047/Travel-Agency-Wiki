OB-SR-01

Flight Search Query Processing and Aggregation

This process document outlines the structured approach for processing and aggregating flight search queries within an OTA platform, focusing on search ranking and personalization.

OB — OTA Online Booking and Search Search Ranking and Personalisation

Process Overview

Trigger User initiates a flight search query through the Expedia Open World platform
Outcome Successful completion produces a personalized list of flights with optimized rankings based on user preferences and availability data.
Systems Expedia Open World Amadeus NDC API Sabre GDS Travelport Snowflake AWS SageMaker

Search response under 800ms Data retrieval accuracy over 95% Data aggregation time under 200ms Personalization accuracy over 80% Search ranking accuracy over 75% User satisfaction score over 4.5/5

Risks & Exceptions

GDS outage impacting availability NDC content gaps Handling high volume of concurrent searches Handling inconsistent data formats Balancing personalization with search relevance Dynamic ranking based on real-time data Ensuring a seamless user experience
